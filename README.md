## Install

To install it is very simple, just download and copy the contents of the themes folder to your `.config/themes` folder.

like this: 

```shell 
git clone https://github.com/neko-night/helix 

cd helix
cp -r themes/* ~/.config/helix/themes 
``` 

Enjoy ;-)
